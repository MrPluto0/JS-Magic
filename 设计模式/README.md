# 设计模式

## 创建型

该模式提供了一种**在创建对象的同时隐藏创建逻辑**的方式，它的主要特点是**将对象的创建与使用分离**。

- 单例模式
- 工厂模式
- 抽象工厂模式
- 建造者模式
- 原型模式

## 结构型

该模式关注**类和对象的组合**，即如何**将类或对象按某种布局组成更大的结构**。

- 适配器模式
- 桥接模式
- 组合模式
- 装饰模式
- 外观模式
- 享元模式
- 代理模式

## 行为型

该模式用于描述**类或对象之间怎样通信、协作共同完成任务，以及怎样分配职责**。

- 访问者模式
- 模板模式
- 策略模式
- 状态模式
- 观察者模式
- 备忘录模式
- 中介者模式
- 迭代器模式
- 解释器模式
- 命令模式
- 责任链模式
